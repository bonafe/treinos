Esse site não guarda o seu plano de treino no código — os dados ficam só no seu navegador. A biblioteca de exercícios (nomes, vídeos, grupos musculares) já vem com o site; só o seu plano (treinos, distribuição semanal) precisa ser carregado aqui, uma vez em cada navegador/aparelho que for usar. Ele fica salvo neste navegador para as próximas visitas, mesmo offline.
